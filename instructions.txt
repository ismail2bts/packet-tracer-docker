Instructions pour utiliser Cisco Packet Tracer 6.2 avec Docker

Notes :
- utilisez un compte root:
  (commande : su) et après entrer votre mot de pass de root
- Assurez-vous d'avoir un environnement graphique (comme Ubuntu avec un bureau).


#1. Installer Docker (une seule fois) :
   sudo snap install docker
   (Redémarrez votre terminal ou votre session après ça.)

#2. Téléchargel'image Packet Tracer (nécessite Internet, une seule fois) :
   docker pull ismail2bts/packet-tracer:6.2

#3. Lancer Packet Tracer (fonctionne hors ligne après le téléchargement) :
   xhost +local:docker
#apres execute ci-dessous

docker run -it --rm \
--network host \
-e DISPLAY=$DISPLAY \
-v /tmp/.X11-unix:/tmp/.X11-unix \
ismail2bts/packet-tracer:6.2
